缺少十六进制数字

Language Message
简体中文 缺少十六进制数字

字符串中的反斜线未转义，\x 之后的字符是不合法的十六进制字符时， 会导致这个异常。

var a = "\x6";
var path = "C:\xls\path";
Browser Message
IE 缺少十六进制数字
Firefox malformed hexadecimal character escape sequence
Chrome (无异常)
Safari (无异常)
Opera (无异常)

字符串中普通反斜线，应该被转义成双反斜线。

var a = "\\x6";
var path = "C:\\xls\\path";
